# ***강의 웹 사이트  1seok2jo***

## Features

1seok2jo에서는 다음과 같은 기능을 제공합니다.
~~~~~~~~~~~~~~~~~~
1. 유저(회원, 강사, 관리자) - 회원가입, 로그인, 장바구니
  -회원 : 회원가입, 로그인, 마이페이지 (프로필, 내 학습, 내 게시판, 내 결제내역), 장바구니, 강의 듣기 , 수강 신청
  -강사 : 강사 신청, 내 강의 게시판, 내 매출 내역 
  -관리자 : 회원 관리, 강의 관리, 게시판 관리 , 매출 관리
2. 강의 - 강의 추가, 삭제, 수정
3. 게시판 - 자유게시판, 공지사항, QNA, 고객센터, 수강평
4. 강의 결제, 환불
~~~~~~~~~~~~~~~~~~
